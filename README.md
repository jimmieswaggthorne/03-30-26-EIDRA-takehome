# Interviews workspace

The runnable app and all frontend code live under **`web/`** — a Next.js restaurant discovery UI.

- **`web/`** — Next.js app (pages, components, styles, config). See [web/README.md](web/README.md) for install, env, and dev commands.
- **`proxy/`** — local API proxy for the upstream restaurant service (still at repo root; required for local dev alongside `web/`).

**Quick start:** install and run **`proxy`** then **`web`** as described in [web/README.md](web/README.md).
